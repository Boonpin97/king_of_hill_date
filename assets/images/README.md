## Recommended Image Specs

- **Format:** JPG or PNG
- **Dimensions:** At least 600x400 pixels (landscape orientation works best)
- **Aspect ratio:** 3:2 or 4:3 recommended
- **File size:** Keep under 500KB for fast web loading
